## 📋 نظرة عامة

Wolf Host هو داشبورد خفيف وآمن لإدارة ومراقبة بوتات تيليغرام (Python و PHP) من مكان واحد.

## ✨ المميزات

| الميزة | الوصف |
|---|---|
| 🔐 **تسجيل دخول آمن** | JWT مع متغيرات البيئة |
| 🐍 **دعم Python** | يكتشف requirements.txt ويثبّت تلقائياً |
| 🐘 **دعم PHP** | يكتشف composer.json ويثبّت تلقائياً |
| ▶️ **تشغيل / إيقاف / إعادة تشغيل** | تحكم كامل بالعمليات |
| 📊 **لوحة إحصائيات** | إحصائيات مباشرة |
| 📜 **سجلات مباشرة** | عبر WebSocket |
| 🔄 **Watchdog** | إعادة تشغيل تلقائية كل 30 ثانية |
| 📦 **رفع ZIP** | رفع كود البوت واستخراجه تلقائياً |
| 🌐 **واجهة عربية** | لوحة تحكم كاملة بالعربية |

## 🔧 الاستخدام

1. ارفع بوتك كـ ZIP
2. أدخل اسم البوت
3. اضغط **رفع وتثبيت**
4. اضغط **تشغيل**
5. شاهد السجلات مباشرة

## 🛡️ الأمان

- حماية من Path Traversal
- JWT للمصادقة
- رفع ZIP فقط (حد 50MB)
- عزل العمليات
